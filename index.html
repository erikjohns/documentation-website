Introduction

Welcome to the developer documentation for the HRDC Maintenance Ticketing System. This system is designed to facilitate the submission, management, and resolution of maintenance tickets for tenants and administrative staff.

Features

The HRDC Maintenance Ticketing System offers several key features to streamline maintenance ticket submission, management, and communication.

User Authentication and Authorization

  • Admin Login/Logout: Administrators can securely log in and out of the system using Firebase authentication.
  • Tenant Login/Logout: Tenants can log in and out of the system using Firebase authentication.
  • Forgot Password: Send a reset email if the password is forgotten.
  • Role-Based Access Control: Access control is implemented based on user roles, distinguishing between administrators and tenants.

Ticket Submission

  • Form for Tenants: Tenants can submit maintenance tickets through a user-friendly form.
  • Attachment Support: Users can attach relevant documents or images to their maintenance tickets. (Coming Soon)
  • Validation: Validation ensures that required fields are filled before submission.

Ticket Management

  • Admin Dashboard: Administrators have access to a dashboard where they can view all open tickets.
  • Filtering and Search: Administrators can filter and search for tickets based on various criteria.
  • Assignment: Tickets can be assigned to specific maintenance staff for resolution. (Coming Soon)
  • Status Tracking: The system tracks the status of tickets, such as open, in progress, or resolved.

Ticket Details

  • Detailed View: Users can view detailed information about each ticket, including description, priority, and status.
  • Comments and Updates: Users can add comments or updates to tickets, facilitating communication throughout the resolution process. (Coming Soon)
  • Timestamps: Timestamps are provided for ticket creation and updates to track the timeline of each ticket. (Coming Soon)

Notifications

  • Email Notifications: Automated email notifications are sent for ticket submission, assignment, and status updates.
  • Email Notification is sent to both user and the admin.
